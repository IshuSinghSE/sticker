We want to be transparent about our data practices:

How We Use Your Information

  • To analyze usage patterns and optimize features

Data Storage and Security

Your privacy is important to us:

  • Application data is stored locally using browser storage
  • We use industry-standard security measures to protect any data we do collect

  • Request correction of any inaccurate information
